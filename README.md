# AWS Cloud Club I2IT

Official website for the AWS Cloud Club at International Institute of Information Technology (I2IT), Pune. A student-led community dedicated to fostering cloud computing culture, experimentation, and professional growth through AWS technologies.

## 🚀 Tech Stack

- **Core**: React (Vite)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion, Three.js (@react-three/fiber)
- **Routing**: React Router DOM

## ✨ Key Features

- **Modern Visuals**: Glassmorphism design with a continuous canvas particle flow.
- **Cinematic Hero**: 3D particle cloud assembly animation on initial load.
- **Interactive Cards**: 3D tilt and spotlight hover effects on all feature and team cards.
- **Magic Bento Layout**: Dynamic, asymmetric grid arrangements for improved visual hierarchy.
- **Optimized Performance**: Hardware-accelerated animations and efficient particle systems.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop viewing.

## 📁 Project Structure

## 🛠️ Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/aws-cloud-club-i2it.git
   cd aws-cloud-club-i2it
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

### Development

Run the development server:

```bash
npm run dev
```

The application will be available at `http://localhost:5173`.

### Production Build

To create a production-ready bundle:

```bash
npm run build
```

The output will be generated in the `dist/` directory.
